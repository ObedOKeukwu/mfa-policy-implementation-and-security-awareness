# Multi-Factor Authentication (MFA) Policy Implementation

## Overview

This project demonstrates the development and implementation of an enterprise Multi-Factor Authentication (MFA) policy designed to reduce the risk of credential theft, phishing attacks, and unauthorized access.

The project was completed as part of CYBER3080 and focuses on security governance, policy development, access control, and security awareness.

---

## Project Objectives

- Develop an enterprise MFA policy
- Define policy scope and enforcement requirements
- Establish security controls for authentication
- Design control testing procedures
- Create an implementation strategy for organization-wide deployment
- Improve protection against credential-based attacks
